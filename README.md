#   Movie App 

Movie App es una aplicacion movil nativa desarrollada en Flutter, diseñada para explorar el catalogo de cine actual.

# Características

Navegacion principal fluida utilizando PageView.builder y AnimatedContainer, donde el fondo se adapta dinamicamente al poster de la pelicula seleccionada.

Integracion nativa con la API publica de The Movie Database (TMDB) para mostrar peliculas en tendencia, recomendaciones y habilitar un motor de busqueda.

Pantalla de detalles que incluye sinopsis, fecha de estreno, sistema de puntuación y el elenco principal.

Redireccion segura a YouTube para ver el trailer oficial utilizando Intents de Android mediante el paquete oficial url_launcher.

Interfaz "Dark Theme" con acentos en color turquesa (#2EFEEA), optimizada para la experiencia del usuario sin sobrecargar la pantalla.

# Enlace de Descarga (APK)
Hemos generado un archivo ejecutable .apk para probar la aplicacion en dispositivos Android, se recomienda Android 11 o superior.


Nota: Habilitar la instalacion desde origenes desconocidos en tu dispositivo para instalar el APK correctamente.
